## Installation

```shell
gem install puma_dynamic_tags
```

or:

```ruby
# Gemfile
gem "puma_dynamic_tags"
```

```shell
bundle install
```

## Usage

```ruby
# in your gem requirements
require "puma_dynamic_tags"
```

```ruby
# config/puma.rb
plugin :puma_dynamic_tags

tag do
  # your dynamic code
end
```
